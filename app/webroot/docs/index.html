  1. The Website
    Intro
    Here are some key definitions of words used in this manual:

    • back-end: any private web pages and content only visible to users and administrators.
    • browser: the application that users and visitors use to view websites.
    • download: any file that is copied from a website.
    • front-end: any public web pages and content.
    • homepage: first page on the website that visitors see.
    • label: A heading on a category menu.
    • link: A clickable-link on a category menu.
    • menu-bar: the navigation bar at the top of back-end webpages.
    • website: where all content (announcements, entries, news, etc.) is stored.
    • web page: pages on website.
    • upload: any file that is imported into the website.
    Front-end
    What visitors see when visiting website.

    The front-end includes all public-facing web pages (i.e. the homepage and other web pages.) Visitors are only able to see the front-end of the website.
    Back-end
    Where users can login to edit website content.

    How to Login:
    1. Go to the website's homepage.
    2. At the bottom of the homepage, click on the link named "Click Here to Login".
    3. Type in username, password and press the enter key.
    How to Save:
    1. When editing something in the back-end, saving is generally possible by clicking on the Save button in the upper-right corner of the browser window.
    How to Logout:
    1. If already logged in, look for the "Logout" item at the right side of the menu-bar.
  2. The Three Pillars
    Announcements
    Time-sensitive information posted on the website's homepage.

    How to Create:
    1. Login.
    2. In the menu-bar go to: Manage → Announcements → New Announcement.
    3. Type out announcement and save.
    How to Edit:
    1. Login.
    2. In the menu-bar go to: Manage → Announcements → View All.
    3. Click on the title of the announcement to be edited.
    4. Edit and save.
    How to Delete:
    1. Login.
    2. In the menu-bar go to: Manage → Announcements → View All.
    3. Click on the checkbox (to the left of the titles,) of the announcement(s) to be deleted.
    4. Click on the "Delete" button in the upper-right corner of the browser window.
    5. A pop-up window should appear; click on the "OK" button.
    How to View (All):
    1. Login.
    2. In the menu-bar go to: Manage → Announcements → View All.
    Entries
    Pages on the website.

    How to Create:
    1. Login.
    2. In the menu-bar go to: Manage → Entries → New Entry.
    3. Type out entry and save.
    How to Edit:
    1. Login.
    2. In the menu-bar go to: Manage → Entries → View All.
    3. Click on the title of the entry to be edited.
    4. Edit and save.
    How to Delete:
    1. Login.
    2. In the menu-bar go to: Manage → Entries → View All.
    3. Click on the checkbox (to the left of the titles,) of the entry (or entries,) to be deleted.
    4. Click on the "Delete" button in the upper-right corner of the browser window.
    5. A pop-up window should appear; click on the "OK" button.
    How to View (All):
    1. Login.
    2. In the menu-bar go to: Manage → Entries → View All.
    How to Feature Entries:
    1. Login.
    2. In the menu-bar go to: Manage → Entries → View All.
    3. Click on the title of the entry to be featured.
    4. On the right-hand side of the edit page, there are two options for featuring entries: "Featured on Homepage" and "Featured on Category Page".
    5. Check desired checkboxes and save.
    Note:
    - Entries featured on homepage will appear near the bottom of the homepage in the default theme.
    - Entries featured on a category page will appear under the main category that the entries are filed under.*

    * Example: imagine there is a category named "Information". When a user clicks on the Information category at the top of the website, any entries featured on this category page will appear here.
    How to Create Categories:
    1. Login.
    2. In the menu-bar go to: Manage → Entries → Categories.
    3. Click on "+ New Category" link at top of page.
    4. This creates an "Untitled Category" that will appear near the bottom of the categories list.
    5. Click on "Rename" and give category desired name.
    6. Then click save.
    How to Edit Categories:
    1. Login.
    2. In the menu-bar go to: Manage → Entries → Categories.
    3. Click the "Rename" link next to the title of category to be edited.
    4. Edit and save.
    How to Delete Categories:
    1. Login.
    2. In the menu-bar go to: Manage → Entries → Categories.
    3. Click on the "Delete" link next to category to be deleted.
    4. A pop-up window should appear; click on the "OK" button.
    How to View Categories (All):
    1. Login.
    2. In the menu-bar go to: Manage → Entries → Categories.

  3. Media
    Files and Folders
    Organizing uploads.

    How to Create (Folders):
    1. Login.
    2. In the menu-bar go to: Manage → Files.
    3. Beneath the menu-bar click on "New Folder".
    4. This creates an "Untitled Folder" at the bottom of the list of files. Rename folder if desired.
    How to Upload (Files):
    1. Login.
    2. In the menu-bar go to: Manage → Files.
    3. Beneath the menu-bar click on "Upload File".
    4. Choose the item to be uploaded and then click on the "Upload" button.
    How to Rename:
    1. Login.
    2. In the menu-bar go to: Manage → Files.
    3. Click on the "Rename" link on the right side of the item to be renamed.
    4. Type in new name and click on the "Save" button.
    How to Move:
    1. Login.
    2. In the menu-bar go to: Manage → Files.
    3. Click on item to be moved and drag item beneath desired folder.
    How to Delete:
    1. Login.
    2. In the menu-bar go to: Manage → Files.
    Images and Attachments
    Extras for entries.

    How to Add Images:
    1. Login.
    2. In the menu-bar go to: Manage → Entries → View All. (Note: this also works for Announcements and News articles as well.)
    3. Click on the title of the entry to be edited.
    4. Beneath the menu-bar click on "Add Image".
    5. That should create a little box underneath the "Images" section near the bottom of the page.
    6. In the box that was created, click on the solitary button to choose an image.
    7. Save entry.
    8. When the entry is saved, the image will be uploaded and included with the entry. (Note: images will appear at the bottom of the public-facing web page for this entry.)
    How to Add Attachments:
    1. Login.
    2. In the menu-bar go to: Manage → Entries → View All. (Note: this also works for Announcements and News articles as well.)
    3. Click on the title of the entry to be edited.
    4. Beneath the menu-bar click on "Add Attachment".
    5. That should create a little box underneath the "Attachments" section near the bottom of the page.
    6. In the box that was created, click on the solitary button to choose a file.
    7. Save entry.
    8. When the entry is saved, the attachment will be uploaded and included with the entry. (Note: attachments will appear at the bottom of the public-facing web page for this entry.)
    How to Delete Images:
    1. Login.
    2. In the menu-bar go to: Manage → Files.
    3. Click on the title of the entry to be edited.
    4. Look near the bottom of the page for the Images section and click "Delete" for each unwanted image.
    5. Save entry.
    How to Delete Attachments:
    1. Login.
    2. In the menu-bar go to: Manage → Files.
    3. Click on the title of the entry to be edited.
    4. Look near the bottom of the page for the Attachments section and click "Delete" for each unwanted attachment.
    5. Save entry.
  4. Management
    Emergency Alerts
    Letting everyone know.

    How to Setup Server:
    1. In a web browser, open up the main website that will act as the Emergency Alerts server.
      (Note: while it is possible to have multiple Emergency Alerts servers, it's usually better if there is only once source of Emergency Alerts.)
    2. Copy the Emergency Alerts RSS link at the bottom of the homepage.
    3. Then Login to the same website.
    4. In the menu-bar go to: Manage → Preferences → Website.
    5. Paste Emergency Alerts RSS link into the "Emergency Alerts Link" box.
    6. Save Changes.
    How to Setup Client:
    1. In a web browser, open up the main website that acts as the Emergency Alerts server.
    2. Copy the Emergency Alerts RSS link at the bottom of the homepage.
    3. Then Login to a different, client website.
    4. In the menu-bar go to: Manage → Preferences → Website.
    5. Paste Emergency Alerts RSS link into the "Emergency Alerts Link" box.
    6. Save Changes.
    How to Create:
    1. Login.
    2. In the menu-bar go to: Manage → Emergency Alerts → New Emergency Alert.
    3. Type out emergency alert and save.
    How to Edit:
    1. Login.
    2. In the menu-bar go to: Manage → Emergency Alerts → View All.
    3. Click on the title of the emergency alert to be edited.
    4. Edit and save.
    How to Delete:
    1. Login.
    2. In the menu-bar go to: Manage → Emergency Alerts → View All.
    3. Click on the checkbox (to the left of the titles,) of the emergency alert(s) to be deleted.
    4. Click on the "Delete" button in the upper-right corner of the browser window.

    How to View (All):

  5. Odds and Ends
    Calendar
    Staying on schedule.

    Note: At this point, calendars are not an integrated part of Quiet Content. An external calendar source, (i.e. a Google Calendar,) is still necessary. And the external source needs to at least have an XML feed source link.
    How to Set Up:
    1. Login.
    2. In the menu-bar go to: Preferences → Website.
    3. Scroll down the page to the "Website Calendar Link" section.
    4. Copy the HTML calendar link from external calendar source (i.e. in Google Calendar,) and paste the calendar link in the "Website Calendar Link" box (if a public-facing HTML calendar link exists.)
    5. Then, copy the XML calendar link from external calendar source (i.e. in Google Calendar,) and paste the calendar link in the "Website Calendar Source" box.
    6. Save Changes.
    Category Menus
    Quick references for website visitors.

    Note: Category menus are only available for main (top level,) entry categories. Once set, these menus appear at the top of the front-end web pages when a user mouses over a category name.
    How to Edit:
    1. Login.
    2. In the menu-bar go to: Manage → Entries → Categories.
    3. Click on "Edit" link next to desired main category.
    4. Click on the "Category Type" list box and choose the Menu option.
    5. Now, labels and links can be added in the area below by clicking on the + Add Label and + Add Link links.
    6. Entry links are most likely going to be links to entries that are within the category that is currently being edited. Accordingly, those in-category entries are automatically listed in select boxes for each link that is added. Sometimes, however, other types of links need to be used. When that is the case, click on the "Have another link to use?" link (found near the bottom of an added Entry Link box.)
    7. When finished with the category menu changes, save.
